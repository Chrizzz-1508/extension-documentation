Discord Webhook

This extension lets you send various things to a Discord webhook

Features

  • Send a simple message to Discord
  • Send a screenshot of your OBS source, scene, or group
  • Send a local file to Discord
  • Send a formatted JSON embed to Discord

Setup

  1. Install the extension. Refer to the SAMMI documentation on how to install an extension.
  2. Make sure to set up a webhook in your Discord via Server Settings > Integrations
  3. From there, select Webhooks and then New Webhook
  4. Select the channel, give the webhook a name and optionally a profile picture
  5. Copy the Webhook URL â€” you'll need it for all commands.
  6. Open the deck and explore the example buttons to see each command in action.

Available Commands

Discord: Message

FieldDescription
Webhook URLMust contain your Discord Webhook URL (in quotes or via variable)
Thread IDOptional. Numeric string for the target thread in a forum channel
MessagePlain text message to send to Discord (Messages over 2000 characters will automatically be split into multiple messages)

Discord: Source Screenshot

FieldDescription
Webhook URLMust contain your Discord Webhook URL (in quotes or via variable)
Thread IDOptional. Numeric string for the target thread in a forum channel
OBS NameName of your OBS connection (default: Main)
Websocket PWLeave blank if no password is set in OBS
Source NameName of the scene, source or group
Note: use /$global.Main.current_scene$/ to capture the active scene
MessageOptional text message to accompany the screenshot

Discord: Upload File

FieldDescription
Webhook URLMust contain your Discord Webhook URL (in quotes or via variable)
Thread IDOptional. Numeric string for the target thread in a forum channel
File PathLocal file path
Use the file picker or type it manually
MessageOptional message to send with the file

Discord: Send Embeds

FieldDescription
Webhook URLMust contain your Discord Webhook URL (in quotes or via variable)
Thread IDOptional. Numeric string for the target thread in a forum channel
BodyJSON-formatted embed body. Use DiscoHook to generate it
