Discord Webhook

This extension lets you send various things to a Discord webhook

If you want to send a message to your Discord from SAMMI, for example when you go live this extension is for you! Check out the features below!

Features

  • Send a simple message to Discord
  • Send a screenshot of your OBS source, scene or group
  • Send a local file to Discord
  • Send a formatted JSON embed to Discord

Setup

  1. Install the extension. Refer to the SAMMI documentation on how to install an extension.
  2. Make sure to setup a webhook in your Discord. This can be done by going to the settings of your server, then Integrations.

    From there, select Webhooks and then New Webhook.
  3. From there you can select the channel that it should post in, give the webhook a name and even add a profile picture for it.
  4. Once completed, be sure to copy the Webhook URL as we'll need it in all the commands.
  5. You can open the deck and click into any of the example buttons to see an example of the associated command.

Using the Commands

  • For each command, the Webhook URL box should contain the URL you generated from the previous setup between quotes or as it is a yellow box, you can have the URL in a variable somewhere and use that variable instead.
  • The Thread ID box is optional and should contain a string of numbers for the thread in a forum. This can be obtained by right clicking the thread and selecting Copy Thread ID.
  • If the Message is over 2000 characters (Discord Message Limit), the extension will automatically split the message up into multiple messages.
  • Discord: Message - Sends a simple plain text message to the webhook.
    • Message - Should contain the text you want to send.
  • Discord: Source Screenshot - Sends a screenshot of a scene, source or group from your OBS.
    • OBS Name - This should be the name of your OBS connection from the SAMMI OBS Connections settings. Leave as 'Main' if you never changed this.
    • Websocket PW - Your websocket password that you use in the OBS Connections settings. Leave blank if you don't have a password set.
    • Source Name - This is the name of your scene, source or group. You can select it from the drop down menu or type in the box.
      Note: You can use /$global.Main.current_scene$/ to take a screenshot of the current scene dynamically.
    • Message - This is an optional message to send with the screenshot.
  • Discord: Upload File - Sends a local file to Discord.
    • File Path - Path to the file on your PC. You can use the file picker to open a window where you can navigate to the file or type the path here manually.
    • Message - This is an optional message to send with the file upload.
  • Discord: Embed - Sends a JSON formatted embed to Discord.
    • Body - This should be a JSON formatted embed body. You can use a site such as DiscoHook to generate the embed JSON and then paste it here.
