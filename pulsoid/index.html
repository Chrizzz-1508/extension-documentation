Pulsoid for SAMMI

Read and react to your heartrate in SAMMI using Pulsoid

This SAMMI extension lets you read and react to your heartrate within SAMMI with any Pulsoid Heart Rate Monitor.
For more information about Pulsoid, please check out their page here.

Setup

  1. Inside SAMMI go to "SAMMI Bridge", select "Install an Extension" and then the "Pulsoid.sef" file
  2. The Pulsoid website will open automatically. Login with your Pulsoid account, get an authorization token, and copy it
  3. Paste the authorization token inside SAMMI and press "OK"

How to use:

  1. Create a new button inside SAMMI with the "Pulsoid" Trigger
  2. Inside the button commands, use "Trigger Pull Data" to access the heart rate values
  3. You can build your own features. Example: update an OBS Text Source using PULSOID.heart_rate
  4. You can also check connection status and current heart rate in the bridge
