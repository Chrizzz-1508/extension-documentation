Pulsoid for SAMMI

Read and react to your heartrate in SAMMI using Pulsoid

This SAMMI extension lets you read and react to your heartrate within SAMMI with any Pulsoid Heart Rate Monitor.
For more information about Pulsoid, please check out there page here.

Setup

  1. Inside SAMMI go to "SAMMI Bridge", select "Install an Extension" and then the "Pulsoid.sef" file
  2. The pulsoid website will open automatically. Login with your Pulsoid account and get an authorization token and copy it
  3. Paste the authorization token inside SAMMI and press "OK"

    4. How to use:

    1. Create a new button inside SAMMI with the "Pulsoid" Trigger
    2. Inside the button commands, you can use a "Trigger Pull Data" command to access the following values of the heartrate event
    3. Create your own features with it.
      Here is a small example on how you can update an OBS Text Source to show your current heartrate.
      This is possible since you can access the heart rate from everywhere by using PULSOID.heart_rate
    4. You can also check out the connection status and the current heartrate inside the bridge
