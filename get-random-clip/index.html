Get Random Clip

Getting a random clip and displaying it in your OBS using SAMMI has never been easier! Check out the features below!

Features

  • Fully customizable request to Twitch including amount, timeframe, min views, cutoff
  • Pre-configured buttons for Raids and '!so @user'
  • Auto setup button for OBS scene and source
  • Ability to add custom actions on clip play
  • SAMMI commands to create your own buttons

Setup

  1. Install the extension via the SAMMI docs
  2. Make sure OBS is connected to SAMMI
  3. The extension auto-runs and creates default buttons
  4. Nest the scene where you want the clip to play

Customization

    You can customize the default values that the extension uses to get clips from Twitch by editing the variables in the SETTINGS button:
Variable Description
Amount Default number of clips to pull (recommended: 100)
Timeframe Maximum age of clips in days (e.g. 365 = past year)
Min_views Minimum view count required for a clip
Cutoff_length Maximum duration (in seconds) the clip is allowed to play
Add your actions in the Play Clip button.
Feel free to delete the default buttons or use custom ones You can also call 'Get Random Clip' and 'Play Clip' manually
