Clipshow

Ever wanted to display your clips when you change to a specific OBS scene like BRB or Starting soon? This extension has you covered! Check out the features below!

Features

  • Fully customizable request to Twitch including; minimum views & max number of days to get clips from
  • Ability to prioritize clips with higher views
  • Set which scenes the clips play on
  • Auto setup button for creating the OBS scene and source needed

Setup

  1. Install the extension. Refer to the SAMMI documentation on how to install an extension.
  2. Ensure OBS is open and connected to SAMMI
  3. The extension should now auto run the setup and will be usable as is with the default values in the SETTINGS button
  4. Nest the scene that the setup creates in whichever scenes you specify in the SETTINGS button

Customization

You can customize the default values used to fetch clips by editing the variables in the SETTINGS button (Customizing any settings requires a SAMMI restart to take effect.):

Variable Description
broadcaster_name Only change this if using a bot account
max_days How old a clip can be (e.g. 365 = past year)
min_views Minimum views a clip must have
scene_names Names of the OBS scenes where clips should play
prioritize_views true = show most viewed clips first; false = random clips above min_views
show_cliptitle true = show clip title in chat; false = do not show title
