Clipshow

Grabs random clips to play on specified scenes in OBS

Ever wanted to display your clips when you change to a specific OBS scene like BRB or Starting soon? This extension has you covered! Check out the features below!

Features

  • Fully customisable request to Twitch including; minimum views & max number of days to get clips from
  • Ability to prioritise clips with higher views
  • Set which scenes the clips play on
  • Auto setup button for creating the OBS scene and source needed

Setup

  1. Install the extension. Refer to the SAMMI documentation on how to install an extension.
  2. Ensure OBS is open and connected to SAMMI
  3. The extension should now auto run the setup and will be usable as is with the default values in the SETTINGS button
  4. Nest the scene that the setup creates in whichever scenes you specify in the SETTINGS button

Customisation

  • You can customise the default values that the extension uses to get clips from Twitch by editing the variables in the SETTINGS button
    • broadcaster_name - This only needs to be changed if you use a bot account
    • max_days - Default value for how old a clip can be (for example 365 => clip is from the last 365 days)
    • Min_views - Default value for how many views a clip needs to have before it is pulled
    • scene_names - This is where you put the names of your scenes from OBS where you want the clips to play
    • prioritize_views - Setting this value to true will prioritize clips with many views. Setting it to false will mean clips play at random as long as they have the minimum views
    • show_cliptitle - Setting this value to true will show the clip title in chat when it plays. Setting it to false will not display the clip title in chat when it plays
