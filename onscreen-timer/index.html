Onscreen Timer

Creates a Timer in OBS that you and your mods can manage with simple chat commands

This extension creates a Timer in OBS that you can customize and manage with chat commands or SAMMI buttons. Check out the features below!

Features

  • Fully customizable OBS Timer
  • Twitch chat commands for you and your mods
  • Extension commands for custom button logic (channel points, etc.)
  • Timer updates live inside the SAMMI Deck

Setup

  1. Ensure OBS is open and connected to SAMMI
    This is necessary before you install
  2. Install the extension. See the SAMMI documentation for how to install extensions.
  3. The setup will create the required scene and sources.
  4. You will be prompted to add the timer to your current scene. You can decline and add it manually later.
  5. You should see a confirmation popup once setup completes.
  6. Use the Settings button in the deck to change scene/source names and toggle full timestamp display.

Twitch Commands

All commands are restricted to the broadcaster and moderators by default.

Accepted time formats: 01:05:30, 5:30, 30, 900

CommandDescription
!timer start *Starts timer with specific time
!timer pausePauses the timer
!timer start OR unpauseResumes the timer
!timer settime *Sets the timer to a specific value
!timer add *Adds time
!timer remove *Removes time
!timer showShows timer in OBS
!timer hideHides timer in OBS

Extension Commands

These can be used inside SAMMI buttons, e.g., for channel points or event triggers.

Timer: Set Timer

FieldDescription

No: Set + pause

FieldDescription

Customization

  • You can edit the TIMER_FINISHED button (label: "Timer is over Add your own Stuff here") to define behavior when timer hits 0.
  • Edit or disable default chat message
  • Choose whether to leave timer visible at 0
