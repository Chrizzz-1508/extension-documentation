Onscreen Timer

Creates a Timer in OBS that you and your mods can manage with simple chat commands

This extension creates a Timer in OBS that you can customise and manage with chat commands or SAMMI commands! Check out the features below!

Features

  • Fully customisable OBS Timer
  • Twitch chat commands that you and your mods can use to manage the timer
  • Extension commands so that you can have things like channel points add time
    • Start
    • Pause
    • Set Time
    • Add/Remove Time
    • Show/Hide Timer
  • Timer is also updated and displayed inside of the SAMMI Deck

Setup

  1. Ensure OBS is open and connected to SAMMI
    This is necessary before you install
  2. Install the extension. Refer to the SAMMI documentation on how to install an extension.
  3. The setup will now run and create the required scene and sources
  4. When it has created the sources you will be asked if you want to add the Timer to the current scene that you are on. You can decline with No and add it manually later by adding the scene as a source to whichever scene you'd like it displayed on.
  5. When the setup has complete, you should see a popup like the below:
  6. You can enter the Settings button in the deck to edit the scene or source name if you ever change it and to change an option for displaying the timer in full.
    If set to True 50 seconds would display as 00:00:50

Twitch Commands

  • There are various Twitch commands that by default are restricted to you, the broadcaster, and your mods.
  • Accepted Time Formats: use colons between the digits.
    E.g. 01:05:30 (1 Hour, 5 minutes, 30 seconds), 5:30 (5 minutes, 30 seconds), 30 (30 seconds), 900 (900 seconds or 15 minutes)
  • !timer start * - Starts the timer with a specific amount of time. For accepted formats, see above.
  • !timer pause - Pauses the timer
  • !timer start OR unpause - Unpauses the timer
  • !timer settime * - sets the time to a specific time. For accepted formats, see above.
  • !timer add * - Adds a specific amount of time to the timer. For accepted formats, see above.
  • !timer remove * - Removes a specific amount of time from the timer. For accepted formats, see above.
  • !timer show - Displays the timer in OBS
  • !timer hide - Hides the timer in OBS

Extension Commands

  • There are various extension commands that you can use within SAMMI to manage the timer. You can add these to buttons you have for channel point triggers, chat triggers, bit/sub/follow triggers etc.
  • There is a button in the deck Extension Commands Examples that you can check out to see examples of all the commands below
  • Accepted Time Formats: use colons between the digits.
    E.g. 01:05:30 (1 Hour, 5 minutes, 30 seconds), 5:30 (5 minutes, 30 seconds), 30 (30 seconds), 900 (900 seconds or 15 minutes)
  • Timer: Set Timer - Sets the time
    • Time - This box is where you enter the time in an accepted format. For accepted formats, see above.
    • Start Timer - Dropdown box where you can select
      • Yes to Set the time, display the timer and then start the timer
      • No to set the time and pause the timer
      • Don't Change to set the time
  • Timer: Add/Remove Time - Adds or removes time from the timer
    • Option - Dropdown box where you can select
      • Add Time to add time
      • Remove Time to remove time
    • Time - This box is where you enter the time. For accepted formats, see above.
  • Timer: Show/Hide - Shows or hides the timer in OBS
    • Option - Dropdown box where you can select
      • Show to show the timer in OBS
      • Hide to hide the timer in OBS
  • Timer: Start/Pause - Starts or stops the current timer
    • Option - Dropdown box where you can select
      • Start to start the current timer
      • Pause to pause the current timer

Customisation

  • There is a button that handles what happens when the timer finishes. The ID is TIMER_FINISHED and in the deck it is called Timer is over Add your own Stuff here
    • You can add whatever you want to happen after the timer hits 0 in here.
    • You can edit the default chat message that is sent when the timer finishes or turn it off
    • You can even choose to continue displaying the timer at 0 seconds
  • You can customise the font and colour of the timer in OBS. Simply visit the scene that was created - Timer Scene by default - and then right click the source - Timer Source by default - select properties and change anything you'd like
    Note: By default the source opacity filter is at 0% so you will have to show the timer first to see it.
