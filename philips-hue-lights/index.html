Philips Hue Light Control

Controlling your Philips Hue Lights remotely has never been simpler with this extension!

Features

  • Check status and settings of a light
  • Modify settings: on/off, brightness, saturation, temperature, hue
  • Guided setup connects your Hue Bridge to SAMMI

Setup

  1. Install the extension. See the SAMMI documentation.
  2. The setup dialogue will launch. You can auto-discover the Hue Bridge or enter the IP manually.
  3. If pairing is successful, this confirmation will appear:
  4. To change the IP address later, edit the phlc.ini file in your SAMMI root folder.

Commands

Get Light Info

This command allows you to fetch the current settings of one of your commands

Example Payload:

{  "hue": 39563.0,
  "sat": 15.0,
  "bri": 254.0,
  "on": 0.0,
  "ct": 230.0,
  "reachable": 1.0,
  "id": 4.0,
  "name": "Desk" }
FieldDescription
IDID of the light
OnIf the specified light is on. 1 = on, 0 = off
ReachableIf the specified light is reachable by the bridge. 1 = reachable, 0 = unreachable
BriBrightness, in range 0 - 254 (0 is not off)
SatSaturation, in range 0 - 254
CtWhite color temperature, 154 (cold) - 500 (warm)
HueHue value (color) in range 0 - 65535

Modify Light

This commands lets you turn your lights on / off and change the settings of the light.
Any field left blank keeps its current value.
WARNING: Do not set White Temp and Hue / Color at the same time.

FieldDescription
LightID or name of the light
StateTurn the light on or off
BrightnessChange the brightness of the light in a range from 0 - 254 (0 is not off)
SaturationChange the saturation of the light in a range from 0 - 254
White TempChange the white color temperature of the light, 154 (cold) - 500 (warm)
Hue / ColorChange the hue value (color) of the light in range 0 - 65535

Bridge

You can also check out all your available lights with their IDs in your SAMMI Bridge.
