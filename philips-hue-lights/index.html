Philips Hue Light Control

Allows you to control your Philips Hue Lights

Controlling your Philips Hue Lights remotely has never been simpler with this extension! Check out the features below!

Features

  • Ability to check the status and settings of a light
  • Ability to modify the settings of a light including; state, brightness, saturation, temperature and hue value
  • Auto setup walks you through connecting the Philips Hue Bridge to SAMMI

Setup

  1. Install the extension. Refer to the SAMMI documentation on how to install an extension.
  2. The extension should now launch the setup dialogue. Follow the instructions to auto discover the Philips Hue Bridge or enter the IP manually

  3. If the pairing process was successful you will see the following popup
  4. If you need to modify the IP of the Philips Hue Bridge, you can edit the phlc.ini file that the setup creates in the root SAMMI folder

Using the Commands

  • You can refer to the example buttons in the deck at any point for examples of both commands
  • Get Light Info - Fetches information about a specific light
    • ID - ID of the light
    • On - If the specified light is on. 1 = on, 0 = off
    • Reachable - If the specified light is reachable by the bridge. 1 = reachable, 0 = unreachable
    • Bri - Brightness, in range 0 - 254 (0 is not off)
    • Sat - Saturation, in range 0 - 254
    • Ct - White color temperature, 154 (cold) - 500 (warm)
    • Hue - Hue value (color) in range 0 - 65535
    • Example Payload: 
                          
{
"hue": 39563.0,
"sat": 15.0,
"bri": 254.0,
"on": 0.0,
"ct": 230.0,
"reachable": 1.0,
"id": 4.0,
"name": "Desk"
}
  • All the available lights are viewable in the SAMMI Bridge
  • Modify Light - Modifies a specific light (You can use the above command to get the exact settings for a color you want)
  • WARNING: Do not try to set the "White Temp" & "Hue / Color" at the same time!
    • Light - ID of the light (using the name instead is optional)
    • State - Change the status of the light
    • Brightness - Change the brightness of the light in a range from 0 - 254 (0 is not off)
    • Saturation - Change the saturation of the light in a range from 0 - 254
    • White Temp - Change the white color temperature of the light, 154 (cold) - 500 (warm)
    • Hue / Color - Change the hue value (color) of the light in range 0 - 65535
    • Any of the above values can be left empty to retain the currently set value
  • The Light option above has a dropdown box that is prefilled with all available lights that are selectable.
