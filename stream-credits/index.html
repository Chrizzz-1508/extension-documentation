Stream Credits

Collects various information during your stream to create movie like end credits

Have you ever wanted a fully customisable end credits for your stream to display your stream stats and thank your subs? Look no further! Check out the features below!

Features

  • Collect the following information:
    • Stream Stats
    • New Followers
    • New First Time Chatters
    • Channel Point Redemptions
    • Raids
    • Subs & Gift Subs
    • Bits
    • Donations
  • Two included extension commands to let you easily add additional information:
    • SC: Create Text - Customisable Text to be shown
    • SC: Create Image - Customisable Image to be shown

Setup

  1. Install the extension. Refer to the SAMMI documentation on how to install an extension.
  2. Ensure OBS is open and connected to SAMMI
  3. The extension should now launch the setup dialogue. Follow the instructions to setup the required scene and HTML file




  4. If the setup was successful, you'll see the following popup:

  5. The process will have created a html file in the root SAMMI folder. This is required for the credits to function correctly

Customisation

  • Everything about the credits is customisable from the things that are shown to the font, font size and font color
  • Check out the Settings button to change what data is shown in the credits
    Setting any of the below to true will display them, and false will not display them
  • Customising any settings requires a SAMMI restart to take effect
    • display_stream_stats - Display information like total subs, chat messages, etc
    • display_followers - Display new followers
    • display_firsttime_chatters - Display firsttime chatters
    • display_channel_point_redemptions - Display a channel point redemption leaderboard
    • display_raids - Display raids
    • display_subs - Display Subs
    • display_bits - Display Bits
    • display_donations - Display Donations (requires you to purchase and install the corresponding extension from Christina. Streamlabs & StreamElements, Ko-Fi)
    • keep_data - The credit data is normally cleared after playing, turn this to true to keep it in the arrays for example for testing and adjusting speed etc
  • You can also enter the Create Credits button in the deck and freely edit any of the specific credit items. More information on how to use the commands below.

Using the Commands

  • SC: Create Text - Allows you to add a text item to the credit list
    • Text - The text that should be shown
    • Font - Any installed font on your PC with a predefined choice to select from
    • Font Size - Text size
    • Color - Color of the text
    • Speed - How fast the text will scroll
    • Text Align - Where the text will align, left/center/right
    • X-Offset - How many pixels should the text be offset, the starting point depends on the text align, for example if you have text align right, the offset will be from the right side of your screen
  • SC: Create Image - Allows you to add an image item to the credit list
    • Image URL - The URL of the image
    • Max Width - The maximum width the image is allowed to have in px
    • Max Height - The maximum height the image is allowed to have in px
    • Speed - How fast the image will scroll
    • X-Offset - How many pixels should the image be offset from the center of your screen
  • By default the extension was made for 1920x1080p but you can also use it for other resolutions, you just might need to adjust the delays and font sizes
