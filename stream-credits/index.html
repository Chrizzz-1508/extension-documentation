Stream Credits

This extension creates fully customizable end credits for your stream to display your stream stats and thank your supporters when you end your stream.

Features

  • Collect the following information:
    • Stream Stats
    • New Followers
    • New First Time Chatters
    • Channel Point Redemptions
    • Raids
    • Subs & Gift Subs
    • Bits
    • Donations
  • Two included extension commands to let you easily add additional information:
    • SC: Create Text - Customizable Text to be shown
    • SC: Create Image - Customizable Image to be shown

Setup

  1. Install the extension. Refer to the SAMMI documentation on how to install an extension.
  2. Ensure OBS is open and connected to SAMMI
  3. The extension should now launch the setup dialogue. Follow the instructions to setup the required scene and HTML file
  4. Enter your password (or leave it empty if you use no password) If you use no password, please make sure that the "Enable Authentication" Box in your OBS Websocket settings is NOT checked.
  5. If the setup was successful, you'll see the following popup:
  6. The process will have created a HTML file in the root SAMMI folder. This is required for the credits to function correctly.
  7. Ensure that you nest the Stream Credits scene in whichever other scenes you'd like to display the credits.

Customization

SettingDescription
display_stream_statsDisplay information like total subs, chat messages, etc
display_followersDisplay new followers
display_firsttime_chattersDisplay first time chatters
display_channel_point_redemptionsDisplay a channel point redemption leaderboard
display_raidsDisplay raids
display_subsDisplay Subs
display_bitsDisplay Bits
display_donationsDisplay Donations (requires extension from Christina: Streamlabs & StreamElements, Ko-Fi)
keep_dataKeep credit data after playthrough (e.g., for testing)

Note: Changing any settings requires a SAMMI restart to take effect.

Using the Commands

SC: Create Text

FieldDescription
TextThe text that should be shown
FontAny installed font on your PC with a predefined choice to select from
Font SizeText size
ColorColor of the text
SpeedHow fast the text will scroll
Text AlignWhere the text will align (left, center, right)
X-OffsetOffset in pixels; depends on alignment. For example, "right" aligns from right edge.

SC: Create Image

FieldDescription
Image URLThe URL of the image
Max WidthThe maximum width in pixels
Max HeightThe maximum height in pixels
SpeedHow fast the image will scroll
X-OffsetOffset from center of your screen

By default the extension is built for 1920x1080p but it can be adjusted for other resolutions by tweaking delays and font sizes.
