Slot Machine

Create your own customisable slot machine in SAMMI for OBS

Creates a fully customisable slot machine that can be played by your twitch chat and displayed in OBS! Check out the features below!

Features

  • Slot Images are fully customisable
  • Rewards/Prizes are also fully customisable
  • Ability to spin all 3 slots or specific slots
  • Ability to add rewards to the prize pool dynamically
  • Ability to change the slot image configuration on the fly
  • Comes fully setup with a bank of images and default settings

Setup

  1. Copy the entire Slots folder into your SAMMI install location
  2. Install the extension. Refer to the SAMMI documentation on how to install an extension.
  3. Ensure OBS is open and connected to SAMMI
  4. The extension will now run the setup and create the required scenes and sources
  5. When it has created the sources you will be asked if you want to add the Timer to the current scene that you are on.
    You can decline with No and add it manually later by adding the scene as a source to whichever scene you'd like it displayed on.
  6. When the setup has complete, you should see a popup like the below:
  7. Enter the Settings button in the deck and customise any settings
    • broadcaster_name - This is your twitch name. Only change this if you use a bot account and want to have it send the chat messages as the bot
    • symbols_only_from_left - Wins only count from left to right, so Diamond Cherry Cherry wouldn't win
    • min_rest_spins - Minimum amount of spins until the slot will completely stop
    • max_rest_spins - Maximum amount of spins until the slot will completely stop

  8. Customise any rewards - the rewards listed here are all setup for the default bank of images and symbols
    • Prize Name - The name of the prize the user will win
    • Symbol Name - The name of the symbol (By default: Cherry, Diamond, Orange, Grapes, Melon)
    • Amount of Symbols - Amount of matching Symbols needed to win this prize
    • Extension Trigger - This is optional. Can be left empty, if it is set, it will automatically trigger an extension trigger with this name when someone gets this

  9. By default all of the slot chat message commands are turned off in the respective Prize buttons in the deck. If you want to turn on any messages for different prizes, you can do so in each button
  10. After you change any settings or rewards, restart SAMMI

Using the Extension

  • After you have setup the extension to your liking, you can use it 'out of the box' by adding any triggers to the Spin the Slot Machine button. By default !spin is enabled for the broadcaster only
  • If you want a more custom setup you can edit the images and symbols used by adding the images to the images folder within the Slots folder that you copied into your SAMMI install location

    The images MUST be in the format <number>_<name>.png

    After you've done that, you can edit the symbol names (these are the <name> you entered above) and prizes associated with them in the Settings button then restart SAMMI
  • You can then create your own buttons to manage the prizes, if you set an extension trigger for them in the Settings button
    You can use the existing Prize buttons in the deck as an example
    • Set an extension trigger
    • Example Payload: 
                      
{
"symbol_1": "Grapes",
"symbol_2": "Diamond",
"symbol_3": "Cherry",
"user_name": "TestUser",
"same_amount": 1,
"matching_symbol": "None",
"prize": "1 Grape"
}
    • You can then trigger pull the payload object from the trigger

      • symbol_1 - Symbol name for Slot 1
      • symbol_2 - Symbol name for Slot 2
      • symbol_3 - Symbol name for Slot 3
      • username - Username of the user who won the prize
      • same_amount - Amount of matching symbols
      • matching_symbol - Symbol name of the matching symbol
      • prize - Prize name entered in the Add Reward command
  • You can then trigger an extension trigger with the name SLOTS MESSAGE * which will automatically wait 7 seconds and then send your specified chat message (so as not to spoil the result from any delay)

Extension Commands

  • The following are extension commands that you can use to further customise how you use the extension. All the below commands have examples in the Extension Commands Examples button in the deck
  • Slots: Play - Sets a user to pass to the Slot machine and starts playing with the default settings (3 slots and stop them with 1s delay in between)
    • User Name - The user name to pass to the slot machine
    • A wait until variable exists command is needed when using this command. You must wait for "SLOTS.is_finished" to be true
  • Slots: Spin - Spins a specified slot or all slots
    • Select Slot - Dropdown menu to select
      • All - Spins all 3 slots
      • Slot 1 - Spins slot 1
      • Slot 2 - Spins slot 2
      • Slot 3 - Spins slot 3
  • Slots: Stop - Stops the selected slot
    • Select Slot - Dropdown menu to select
      • All - Stops all 3 slots
      • Slot 1 - Stops slot 1
      • Slot 2 - Stops slot 2
      • Slot 3 - Stops slot 3
    • Force Stop - Dropdown menu to select
      • Yes - Immediately stops the slot
      • No - Gradually stops the slot after 15-20 symbol changes
  • Slots: Set Slot - Set the slot to any symbol of your choice (must be in the images directory and loaded)
    • Select Slot - Dropdown menu to select
      • All - Selects all 3 slots
      • Slot 1 - Selects slot 1
      • Slot 2 - Selects slot 2
      • Slot 3 - Selects slot 3
    • Value - The name of the Symbol
  • Slots: Show/Hide - Fades the slots in and out
    • Option - Show or Hide the Slots
      • Show - Fades the slot in
      • Hide - Fades the slot out
    • User Name - This is optional. A username you can pass to be displayed on the slot machine interface. If left empty no username is sent.
  • Slots: Add Reward - Adds a reward to the prize pool
    • Prize Name - The name of the prize the user will win
    • Symbol Name - The name of the symbol
    • Amount of Symbols - Amount of matching Symbols needed to win this prize
    • Extension Trigger - This is optional. Can be left empty, if it is set, it will automatically trigger an extension trigger with this name when someone gets this
  • Slots: Change Image Array - Changes the images for the slots. Can be left empty to reset to the default slots
    • Image Names - This is a list of images names split by a semi colon (;)
    • Example list: 2_Diamond.png;7_Melon.png;2_Diamond.png;7_Melon.png;8_Orange.png
