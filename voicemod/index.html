Voicemod for SAMMI

Effortlessly transform your voice during livestreams using Voicemod and SAMMI

This SAMMI extension integrates with Voicemod, allowing you to dynamically control and change your voice in real time. It adds an interactive, fun layer to your streams or content creation.

Note: Use version V3 only if you are using Voicemod V3 or higher. If you're using an older version, please use the V1.1 extension instead.

Setup

    Inside SAMMI go to "SAMMI Bridge", select "Install an Extension" and then the "Voicemod.sef" file

SAMMI Commands

This extension provides several useful commands to control Voicemod directly from SAMMI. These commands may take some time to execute—use Wait Until Variable Exists when needed.

Command NameDescription
VM: Change to Specific VoicePick and switch to a voice of your choice.
VM: Random VoiceLet SAMMI surprise you with a random voice.
VM: Reset VoiceRevert back to your natural voice.
VM: Toggle Background EffectsTurn on or off any background effects.
VM: Toggle Hear My VoiceControl whether you can hear your own voice transformation.
VM: Toggle MuteMute or unmute your voice.
VM: Toggle Voice ChangerEnable or disable the voice changer feature.
VM: Get Voice BitmapFetch the image for the selected voice.
VM: Prevent Bad LanguageHelps filter inappropriate language using mute/beep for x milliseconds.

SAMMI Triggers

The following trigger is available to react when a voice change occurs:

Trigger NameDescription
Voicemod: Voice ChangedTriggered whenever the voice is altered in any way.

Bridge Interface

The bridge provides real-time info about Voicemod's state:

FeatureDescription
StatusIndicates whether you’re connected to Voicemod.
VoicechangerShows the current voice setting.
Background EffectsDisplays the status of background effects.
Hear My VoiceLets you know if you can hear your transformed voice.
MutedIndicates if your voice is muted.
License TypeDisplays the type of Voicemod license you have.
Voice IDShows the identifier for the current voice.
